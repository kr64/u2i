Welcome to u2i

Simple Arduino-based USB/serial to I2C/SMBus bridge.

New Features

v0.4 29/12/2013 Status byte added (command z)

Commands

  • a [a..] view/set i2c address(es)
  • f [speed] view/set bus frequency in kHz (kb/s)
  • h show help
  • r [c] [b] smbus read byte/word c=command b=nofbytes_
  • s [a0] [a1] scan i2c bus from address a0 to a1
  • v show version
  • w [c] [b..] smbus write byte(s) c=command
  • z [v] view/set status byte

Notes:

  1. all values to u2i in hex w/o format specified
  2. smbus write commands will be grouped by default

Supported Arduino platforms

  1. Arduino Micro
  2. Arduino Leonardo
